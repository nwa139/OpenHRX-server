Use OpenHRX for you!

OpenHRX is free and easy to run, you only need to install the OpenHRX Server and a database (e.g. PostgreSQL). If you have less than 500 employees to manage, running both on a good computer is generally more than enough.
Even if you know little about computer, the steps are easy to follow. Follow this link to get a detailed tutorial about how to use OpenHRX in your enterprise
